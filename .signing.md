## Git Commit Signing Setup (SSH‑based, GitHub‑verified)

```
sudo apt install git gh -y
```

## Log in

```
gh auth login
```

## Generate or locate your SSH signing key

```
gh ssh-key list
```

## (Optional) Generate Key

```
gh ssh-key generate --type signing
```

## Upload Key

```
gh ssh-key add ~/.ssh/id_signing.pub --type signing
```

## Configure Git to use SSH signing

```
git config --global gpg.format ssh
git config --global gpg.ssh.program "$(which ssh-keygen)"
git config --global user.signingkey "<your-signing-key-id>"
git config --global commit.gpgsign true
```

## Signing Key Verification

```
gh ssh-key list
```

## Start SSH agent and load your signing key

```
eval "$(ssh-agent -s)"
```

Add your private signing key:

```
ssh-add ~/.ssh/id_signing
```

Or, load existing key

```
ssh-add ~/.ssh/id_ed25519
```

## Create the allowed signers file (required for local verification)

```
mkdir -p ~/.config/git
nano -cw ~/.config/git/allowed_signers
```

Add this line (replace email + key):

```
<your-github-email> <your-public-signing-key>
```

Tell Git where it is:

```
git config --global gpg.ssh.allowedSignersFile ~/.config/git/allowed_signers
```

## Test signing

```
git commit --allow-empty -m "Signed test"
git log --show-signature -1
```

## (Optional) Auto‑load signing key at login

```
eval "$(ssh-agent -s)" >/dev/null
ssh-add ~/.ssh/id_signing 2>/dev/null
```
